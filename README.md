# 📊 Customer Churn Analysis Dashboard

## 🔹 Overview  
This project analyzes customer churn behavior to identify key drivers and quantify business impact.

## 🎯 Objectives  
- Calculate churn and retention rate  
- Identify high-risk customers  
- Analyze churn drivers  
- Estimate revenue loss  

## 🛠️ Tech Stack  
- SQL  
- Power BI  
- DAX  

## 📂 Dataset  
- ~50,000 customer records  
- Features: engagement, recency, purchase frequency, spend  

## 📈 Key Metrics  
- Churn Rate: 28.9%  
- Revenue Lost: INR 22.65M  
- Revenue at Risk: INR 3.6M  

## 🔍 Key Insights  
- High inactivity (>90 days) → ~50% churn  
- Low engagement → higher churn  
- High abandonment → increased churn  

## 📊 Dashboard Features  
- KPI cards  
- Segmentation analysis  
- Risk-based customer analysis  
- Interactive filters  

## 🚀 Business Impact  
Helps identify high-risk customers and reduce revenue loss through targeted retention strategies.
